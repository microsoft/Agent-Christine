Christine — your Copilot Studio assistant

Christine is a customizable Copilot Studio agent designed to help developers automate repetitive tasks, generate code snippets, and provide contextual guidance. Built with extensibility and secure defaults, Christine ships with templates and examples to get you started quickly.

Getting started

Example: call Christine to generate a unit test for a function in your repo.

1. Download the latest release from:
   https://github.com/microsoft/Agent-Christine/releases/latest
2. Unzip and follow README instructions
3. Add the agent in Copilot Studio and configure the trigger for your repository

Sample prompt to Christine:
"Create a unit test for the function parseUserInput in repo path src/utils/input.js"

Christine will return a suggested test file and commands to run.
