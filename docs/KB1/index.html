Executive Summary: Agent Christine is an AI agent developed by the Marketplace Fastrack team to help SDCs Marketplace journey by integrating organizational data with public documentation for streamlined knowledge retrieval and workflow automation. It offers targeted instructions for rapid agent creation, supporting onboarding and self-sufficiency for finance teams, sales, and admins. This AI agent solution leverages FastTrack best practices and enables continuous learning and customization through collaboration with engaged partners. Distribution via GitHub and future Marketplace integration ensure broad accessibility and adaptability.

       Purpose: Agent Christine is to act as a trusted advisor for users seeking guidance on Microsoft Commercial Marketplace topics. This document is a knowledge source to provide clear, concise, and reliable answers by referencing curated knowledge sources first and web search only when necessary.

 

Definitions

  • Publisher = Software Development Company (SDC)
  • MSX Proposal = digital quote in MSX (not Quote Center ).
  • MCA-E = Microsoft Customer Agreement Enterprise.
  • EA = Enterprise Agreement.
  • SCE = Server & Cloud Enrollment (EA Azure).
  • Legacy → New Commerce migration = EA → MCA.
  • MACC = Microsoft Azure Consumption Commitment.
  • MEMACC = Multi-Entity MACC.
  • CAPT = Customer Affiliate Purchase Terms.
  • MBSA = Predecessor to MCA.
  • Billing Account = top-level financial entity.
  • Billing Profile = sub-entity for payments/discounts/reservations.
  • ACR = Azure Consumed Revenue.
  • ACD = Azure Commitment Discount.
  • ACO = Azure Credit Offer (not a discount).
  • Quote Center = MSX Proposals
  • TACO / CACO = Time-based / Consumption-based ACOs.
  • M365 / D365 = per-user (license-based) products.
  • Step-up SKU = Component upgrade SKU in MCA-E.
  • MP= Marketplace purchase
  • PO=Private offer
  • ISV=independent software vendor
