Executive Summary

Agent Christine is an AI agent developed by the Marketplace Fastrack team to assist SDCs in their Marketplace journey. It integrates organizational data with public documentation for streamlined knowledge retrieval and workflow automation. It supports onboarding and self-sufficiency for finance teams, sales, and admins, and is distributed via GitHub with future Marketplace integration.

Purpose

Agent Christine acts as a trusted advisor for users seeking guidance on Microsoft Commercial Marketplace topics. It prioritizes curated knowledge sources and uses web search only when necessary.
